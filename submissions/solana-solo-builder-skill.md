# solana-solo-builder-skill

**Author**: Mayahang  
**Repo**: https://github.com/Mayahang/solana-solo-builder-skill  
**Type**: Novel skill  

## What it solves

The complete lifecycle skill for solo Solana hackathon builders - from idea scoping to live submission. Covers secrets management, free tier deployment, bonding curve token launchpads, anti bot protection, and winning README writing.

## Sub-skills

- `scope.md` — hackathon idea selection, deadline planning
- `secrets.md` — API key safety, git history rewrite, incident response
- `deploy.md` — Vercel, Wispbyte, keepalive patterns
- `launchpad.md` — bonding curves, SPL tokens, decimal debugging
- `anti-bot.md` — rate limiting, wallet age checks, purchase caps
- `submit.md` — README structure, judging criteria
- `debug.md` — RPC errors, PDA bugs, instruction data errors
